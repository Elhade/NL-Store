## Catalogue produits NL Store

### Catégories WooCommerce (slugs utilisés dans le code)

| Slug | Label affiché | Sous-items template accueil |
|---|---|---|
| `bebe` | Bébé | Biberons, Couches, Lingettes |

| `vetements` | Vêtements | Bébé, Enfant |
| `hygiene` | Hygiène | Crèmes, Lingettes, Produits bébé |

Images fallback catégories (si pas d'image WooCommerce) : `assets/imgs/cat-{slug}.jpeg`.

### Points d'attention visuels

- Les images produits sources ont des bandeaux dorés incrustés (ex : « AÏSHA », « BACCARAT »). Ces visuels ne sont pas propres pour la boutique WooCommerce.
- Les images de catégories dans WooCommerce sont à uploader (les fallback thème sont des photos génériques).

**Why:** Ces informations de catalogue sont nécessaires pour configurer correctement les produits WooCommerce, les images et les textes du template accueil.
